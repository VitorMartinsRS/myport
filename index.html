Projeto de desenvolvimento front-end

Projeto desenvolvido por estudantes dos cursos de ti com o intuito de aprimorar os conhecimentos de front-end

O projeto tem como objetivo o desenvolvimento de um site com foco no front-end, utilizando as principais ferramentas aprendidas em aula. Foram aplicados conceitos de HTML, para estruturar o conteúdo da página; CSS, para a estilização dos elementos visuais; e JavaScript, com manipulação do DOM e uso de JSON, possibilitando interatividade e organização de dados.

A proposta busca reforçar os conhecimentos adquiridos, promovendo uma aplicação prática que estimula o raciocínio lógico, a criatividade e o desenvolvimento de habilidades técnicas. Com isso, o aluno ganha mais experiência no uso das tecnologias e constrói uma base sólida para projetos futuros no campo do desenvolvimento web.
